# 富文本编辑器

这篇主要介绍一下富文本编辑器的配置，原本的富文本编辑器没有字体和字号的设置，并且原本的富文本编辑器在上传图片时，图片的路径会自动转化为相对路径，影响前端图片的显示。

  1. 目录结构
├── src    
│   ├── components                 # 模板组件 
│   │   ├── Tinymce                # 富文本编辑器
│   │   │   ├── index.vue          # 入口文件  
│   │   │   ├── toolbar.js         # 配置工具栏
  • index.vue文件：文件包含富文本编辑器的一些配置。
  • toolbar.js文件：富文本编辑器中工具栏的配置。
  1. 优化的配置项

  • 设置字体和字号

    设置字体和字号具体要改的文件涉及到index.vue和toolbar.js：

    在index.vue文件里面新增fontsize_formats配置，如下所示：

    window.tinymce.init({
  convert_urls: false, // 新增的配置 禁止图片路径转化为相对路径
  language: this.language,
  selector: `#${this.tinymceId}`,
  height: this.height,
  body_class: 'panel-body ',
  object_resizing: false,
  toolbar: this.toolbar.length > 0 ? this.toolbar : toolbar,
  fontsize_formats: '8px 10px 12px 14px 16px 18px 20px 22px 24px 36px', // 新增的配置 字体大小
  menubar: this.menubar,
  plugins: plugins,
  end_container_on_empty_block: true,
  powerpaste_word_import: 'clean',
  code_dialog_height: 450,
  code_dialog_width: 1000,
  advlist_bullet_styles: 'square',
  advlist_number_styles: 'default',
  imagetools_cors_hosts: ['www.tinymce.com', 'codepen.io'],
  default_link_target: '_blank',
  link_title: false,
  nonbreaking_force_tab: true,
  init_instance_callback: editor => {
    if (_this.value) {
      editor.setContent(_this.value)
    }
    _this.hasInit = true
    editor.on('NodeChange Change KeyUp SetContent', () => {
      this.hasChange = true
      this.$emit('input', editor.getContent())
    })
  },
  setup(editor) {
    editor.on('FullscreenStateChanged', (e) => {
      _this.fullscreen = e.state
    })
  }
})

    在toolbar.js文件中新增fontsizeselect和fontselect配置，整体配置如下所示：

    const toolbar = 
[
  'searchreplace bold italic fontsizeselect fontselect underline strikethrough alignleft aligncenter alignright outdent indent  blockquote undo redo removeformat subscript', 'hr bullist numlist link image charmap preview anchor pagebreak insertdatetime media table emoticons forecolor backcolor fullscreen superscript code codesample'
]

    配置好以上两点，字体和字号的设置就配置好了。

    • 上传图片的路径的修改

      将convert_urls: false配置加上，会禁止将图片路径转化为相对路径。
