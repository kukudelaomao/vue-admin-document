# 国际化

  1. 目录介绍

    ├── src
│   ├── components                 # 组件
│   │   ├── LangSelect             # 语言选择组件                       
│   ├── lang                       # 语言文件夹
│   │   ├── zh.js                  # 中文
│   │   ├── en.js                  # 英文
│   │   ├── index.js               # 入口文件
├── main.js                        # 配置国际化
    • LangSelect：语言选择组件。
    • zh.js 文件：提供中文。
    • en.js 文件：提供英文翻译。
    • index.js 文件：引入zh.js和en.js文件，并设置默认语言。
    • main.js 文件：引入国际化。

  2. 语言包文件及默认语言

    zh.js 和 en.js文件：

    export default { 
  route: {
    dashboard: '首页',
    ···
  }   
  ···
}

    export default { 
  route: {
    dashboard: 'Dashboard',
  }
  ···
}

    main.js文件：

    import Vue from 'vue'
// 引入VueI18n为了国际化
import VueI18n from 'vue-i18n'
// 引入Cookies存入用户设置的语言
import Cookies from 'js-cookie'
// 引入element语言包
import elementEnLocale from 'element-ui/lib/locale/lang/en' 
import elementZhLocale from 'element-ui/lib/locale/lang/zh-CN'
// 引入本地语言包
import enLocale from './en'
import zhLocale from './zh'

Vue.use(VueI18n)

const messages = {
  en: {
    ...enLocale,
    ...elementEnLocale
  },
  zh: {
    ...zhLocale,
    ...elementZhLocale
  }
}

// 获取语言
export function getLanguage() {
  // 如果Cookies里面有用户之前存入的语言，就用之前用户选择的语言
  const chooseLanguage = Cookies.get('language')
  if (chooseLanguage) return chooseLanguage

  // 获取浏览器的默认语言，如果本地的语言包不包含默认语言，则返回英文，如果存在，则返回本地的默认语言
  const language = (navigator.language || navigator.browserLanguage).toLowerCase()
  const locales = Object.keys(messages)
  for (const locale of locales) {
    if (language.indexOf(locale) > -1) {
      return locale
    }
  }
  return 'en'
}
// 应用国际化
const i18n = new VueI18n({
  // set locale
  // options: en | zh | es
  locale: getLanguage(),
  // set locale messages
  messages
})

export default i18n

    main.js 文件：

    Vue.use(Element, {
  size: Cookies.get('size') || 'medium', // set element-ui default size
  i18n: (key, value) => i18n.t(key, value)
})

  3. .vue文件中应用国际化

    • 页面模板中应用：

      {{ $t('route.dashboard') }}

    • 属性中应用：

      :label=$t('route.dashboard')

    • js中应用：

      this.$t('route.dashboard')

在src/components/LangSelect文件夹里面有语言选择组件，如下图所示：

 <el-dropdown-menu slot="dropdown">
  <el-dropdown-item :disabled="language==='zh'" command="zh">
    中文
  </el-dropdown-item>
  <el-dropdown-item :disabled="language==='en'" command="en">
    English
  </el-dropdown-item>
</el-dropdown-menu>
