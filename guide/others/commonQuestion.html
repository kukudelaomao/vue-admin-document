# 常见问题

# webpack中alias的配置

在vue.config.js文件中，存在代码：

resolve: {
  alias: {
    '@': resolve('src')
  }
}

这行代码的意思是用@来指代src路径。所以在引入组件时，可以采用import pagination from '@/components/until/pagination'其中@/components/until/pagination是指src文件夹下的/components/until/pagination。

# npm install 与 cnpm install

采用npm install可能会有node-sass安装失败的问题，此时可以用cnpm install进行安装，虽然官方不建议用cnpm install进行安装，但是经实践采用cnpm install目前并没有发现问题。
